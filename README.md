# ESP32 Marauder DIY

> **Construye tu propio dispositivo basado en ESP32 Marauder desde cero, entendiendo el proceso y evitando los errores más comunes.**

## ¿Qué es este proyecto?

Este repositorio documenta mi proceso para construir un dispositivo compatible con **ESP32 Marauder** utilizando componentes comprados por separado.

La idea es sencilla: en lugar de adquirir un dispositivo ya ensamblado, construirlo pieza a pieza, aprendiendo durante el proceso y documentando todo lo necesario para que cualquier otra persona pueda hacer lo mismo.

No pretende sustituir la documentación oficial del proyecto ESP32 Marauder, sino complementarla con una experiencia práctica de montaje.

---

# Objetivos

* Documentar todo el proceso de construcción.
* Explicar qué componentes hacen falta y por qué se han elegido.
* Mostrar los errores encontrados y cómo solucionarlos.
* Facilitar que cualquier persona pueda reproducir el montaje.
* Mantener una documentación clara, ordenada y actualizada.

---

# ¿Qué encontrarás aquí?

* 📦 Lista completa de materiales.
* 🔧 Guía de montaje paso a paso.
* 💻 Instalación del firmware.
* ⚙️ Configuración inicial.
* 📷 Fotografías del montaje.
* 🚨 Problemas habituales y soluciones.
* 📝 Notas y mejoras realizadas durante el proyecto.

Todo el contenido estará basado en una experiencia real de construcción. Si algo falla, también quedará documentado.

---

# Filosofía del proyecto

Este repositorio está escrito desde el punto de vista de alguien que está construyendo el dispositivo, no desde el de quien ya lo conoce todo.

No encontrarás explicaciones innecesariamente complejas cuando una explicación sencilla sea suficiente.

La prioridad es que puedas seguir el proceso, entender qué estás haciendo y terminar con un dispositivo funcionando.

Cuando aparezcan problemas, también se documentarán. Los errores forman parte del proyecto y, muchas veces, son la información más útil para quien viene detrás.
